Stalk.io

a social web chat service for your web pages.

Just insert some tags in your HTML pages.
It only takes a minute.

How to install this on your web pages ? click this !!



If you already set up this on your web page,
Move to the Dashboard page.

How to install ?

Put the tags and the javascript codes before the last </body> tag in your HTML page

<script type="text/javascript" src="http://www.stalk.io/stalk.js" charset="utf-8" ></script>
<script language="javascript">
   STALK.init();
</script>

That's it! :)
